# Clinic Management System (Java OOP Project)

## Overview
A simple console-based Clinic Management System built using Java and Object-Oriented Programming (OOP).  
The system allows basic patient management through a menu-driven interface.

---

## Features
- Add new patients
- View all patients
- Search patient by ID

---

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList (data storage)
- Scanner (user input)

---

## Project Structure
- Patient.java → Model class (stores patient data)
- PatientService.java → Handles logic (add, view, search)
- Main.java → User interface and menu system

---

- Separation of concerns (OOP design)
